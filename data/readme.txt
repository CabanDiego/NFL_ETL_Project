The Data file used for this project is too large to upload to github, to
successfully run the project download the csv file in the following link:
https://www.kaggle.com/datasets/maxhorowitz/nflplaybyplay2009to2016?resource=download
Extract the "NFL Play by Play 2009-2016 (v3)" file from the folder and place into this data/ folder 
and rename it "nfl_play_by_play.csv"
For Testing purposes only a small selected amount of rows was used.
